Power BI Task 1: Financial Health Dashboard
 Project Title:-
Financial Health Dashboard using Power BI
 Domain:
Business Intelligence / Data Analytics
 Objective:-
The objective of this project is to develop an interactive Power BI dashboard that analyzes an organization’s financial health by visualizing income statements, balance sheets, and cash flow data. The dashboard provides real-time insights to support budgeting, forecasting, and strategic decision-making, especially for SMEs.
✓ Problem Statement:-
Organizations often struggle to track financial performance across multiple reports and time periods. This project consolidates financial data into a single dashboard that enables stakeholders to:
Monitor profitability trends
Understand revenue and expense patterns
Analyze cash flow stability
Forecast future financial performance
1. Dataset Description
The dataset includes:
Income Statement data (Revenue, Expenses, Profit)
Balance Sheet data (Assets, Liabilities, Equity)
Cash Flow data (Operating, Investing, Financing activities)
Time-based financial records (Monthly/Yearly)
Data source used: Sample financial dataset (CSV/Excel)
✓ Tools & Technologies Used
Power BI Desktop
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs
Excel / CSV – Data source

 Key Features & Requirements Implemented
